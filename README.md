# 一打目 — 麻雀牌効率ドリル

門前・中盤・副露後の打牌を扱う麻雀牌効率の練習サイトです。100問の検証済み問題バンクから、シチュエーションと難易度を混在させて毎回10問をランダム出題します。

## 前提

- ドラ・場況・点棒状況は考慮しない
- 鳴くかどうかの判断は考慮しない（副露後の手牌は出題対象）
- 受け入れと将来の変化を重視
- 解答は標準的な牌効率ベースの模範解答

## ローカル起動

静的サイトなので、任意のHTTPサーバーで公開できます。

```bash
node scripts/serve.mjs
```

問題データの牌数・重複・受け入れは次で検査できます。

```bash
node scripts/validate.mjs
```

## 構成

- `index.html` — ページ構造
- `styles.css` — レスポンシブデザイン
- `app.js` — 問題データとクイズ動作
- `questions.generated.js` — 自動生成・検証済みの100問
- `scripts/generate-questions.mjs` — 問題バンク生成
