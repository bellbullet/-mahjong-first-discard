# 一打目 — 麻雀牌効率ドリル

門前手の第一打に特化した、麻雀牌効率の練習サイトです。

## 前提

- ドラ・場況・点棒状況は考慮しない
- 鳴き判断は考慮しない
- 受け入れと将来の変化を重視
- 解答は標準的な牌効率ベースの模範解答

## ローカル起動

静的サイトなので、任意のHTTPサーバーで公開できます。

```bash
node scripts/serve.mjs
```

問題データの牌数・重複・受け入れは次で検査できます。

```bash
node scripts/validate.mjs
```

## 構成

- `index.html` — ページ構造
- `styles.css` — レスポンシブデザイン
- `app.js` — 問題データとクイズ動作
